سلام برای راه اندازی پروزه از دستورات زیر استفاده کنید 1- npm run dev 2-npm run json-server در صورت صحت اطلاعات وارد شده در فرم درخواست با موفقیت ارسال خواهد شد.. db.json تمامی اطلاعات ارسالی در این فایل ذخیره میشوند همه منطق پروژه و کدهای آن در یک کامپوننت نوشته شده.. البته بهتر است که برای سهلوت کار و اصول کلین کد به چندین کامپوننت مجزا تقسیم شود ولی چون هدف فقط پیاده سازی تسک محول شده بود فقط در یک کامپوننت انجام شد. تمامی استایل های پروژه در یک فایل استفاده شده موفق و موید باشید پوریا مبارکی
